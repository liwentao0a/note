### Markdown Preview Github Styling

### Markdown Image

一个用于方便的在 Markdown 中插入图片的扩展，支持将图片存放在本地或第三方的图床或对象存储。

可复制图片文件或截图粘贴，快捷键 Ctrl + Shift + V，或右键菜单粘贴图片。
